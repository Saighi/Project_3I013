Nous avons codé l'application dans un repertoire Project_3I013
Si vous l'installez dans un répertoire avec un autre nom, changer la constante ROOT_FOLDER dans /class/Tools.php
(Si installé dans la racine, mettre " / ")


Le Fichier CSS pour certains aspects design est hebergé dans le serveur de bootstrap.com, il est donc accesible qu'en étant connecté à internet.
Si travail constant sans connexion internet envisagée, le fichier CSS peut être téléchargé sur le site de bootstrap.
